# Integração SQL Server e Excel (AdventureWorks)

Este projeto demonstra a integração entre o banco de dados SQL Server e o Microsoft Excel para automação de relatórios e análise de dados.

## 🛠️ Tecnologias Utilizadas
* **SQL Server**: Consultas estruturadas para extração de dados.
* **Excel**: Front-end para visualização e relatórios dinâmicos.
* **Database**: AdventureWorks (Sample Data).

## 🚀 O que este projeto faz?
1. Extrai dados de vendas através de scripts SQL otimizados.
2. Integra os resultados diretamente no Excel para criação de Dashboards.
3. [OPCIONAL] Conecta com serviços AWS para armazenamento/análise em nuvem.

## 📁 Estrutura dos Arquivos
* `Passo a Passo.sql`: Scripts de consulta utilizados.
* `Relatório de Vendas`: Planilha final com os dados integrados.
